# デモ台本集

> 研修・営業・社内布教の場で使う before/after デモの台本を蓄積。

## ファイル構成

各デモは以下の構成で保存:

```
demo-XXX-[タイトル].md
```

## 含めるべき要素

- 想定シーン（どんな業務か）
- before（AI を使わない場合の所要時間・手順）
- after（AI を使った場合の所要時間・手順）
- 実演プロンプト
- 想定される反応・質問への返し方

## デモテンプレート

```markdown
# Demo: [タイトル]

## 想定シーン
（誰が・いつ・どんな状況で）

## Before（AIなし）
- 所要時間: XX分
- 手順:
  1.
  2.

## After（AIあり）
- 所要時間: XX分
- 手順:
  1. プロンプトを投げる
  2. レビュー・修正
  3. 完成

## 実演プロンプト

\`\`\`
（プロンプト本文）
\`\`\`

## 想定される質問・返し
- Q: 「結局チェックする時間が必要では？」
  A: 「ゼロからより、修正のほうが速い」点を実演
- Q: 「機密情報はどうする？」
  A: 匿名化ルールの紹介
```

## 推奨デモ（最初に揃えるべき5本）

1. 議事録要約（5分→30秒）
2. 提案メール作成（30分→5分）
3. 社内マニュアル下書き（半日→30分）
4. アンケート集計（2時間→10分）
5. 多言語メール（30分→2分）
