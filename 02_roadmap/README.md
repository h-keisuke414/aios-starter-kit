# 6ヶ月導入ロードマップ

## 全体フロー

```
M1 Kickoff      → 環境構築・初期トレーニング・ベースライン測定
M2 Pilot        → 1ユースケース試験運用
M3 Stabilize    → 定常運用フロー定着・社内マニュアル化
M4 Expand       → 第2・第3ユースケース展開
M5 Optimize     → データ検証・プロンプト改善
M6 Review       → 成果報告・次期契約設計・内製化引き継ぎ
```
