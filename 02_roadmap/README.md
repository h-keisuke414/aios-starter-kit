# 6 ヶ月導入ロードマップ

中小企業向け AI 導入の標準フロー。
**1 ヶ月で全部やろうとせず、小さな成功 → 信頼 → 拡張** の順で進める。

---

## 全体フロー

```
M0 Discovery    → 地域リサーチ・初回ヒアリング・戦略合意（契約前 or 契約直後）
M1 Kickoff      → 環境構築・初期トレーニング・ベースライン測定
M2 Pilot        → 1 ユースケース試験運用
M3 Stabilize    → 定常運用フロー定着・社内マニュアル化
M4 Expand       → 第 2・第 3 ユースケース展開
M5 Optimize     → データ検証・プロンプト改善
M6 Review       → 成果報告・次期契約設計・内製化引き継ぎ
```

## 月次運用テンプレート構造

各月のファイルは以下の構成で書かれている:

- **ゴール**: その月の終わりに到達したい状態
- **実施タスク**:
  - 導入支援側（IZAYOI）が行うこと
  - クライアント側が行うこと
- **成果物**: その月で完成させる文書・ツール・実装
- **レビューポイント**: 月末に確認する指標・観察点
- **翌月への引き継ぎ事項**: 次月キックオフで必ず触れる項目

## 進行が遅れた時の対処

| 遅れ方 | 対処 |
|---|---|
| **M1 で AI ログイン進まない** | 個別 1on1 を 1 名 15 分で実施。PC 苦手な人ほど早く拾う |
| **M2 でユースケース選定が決まらない** | 一旦 1 つに絞って動かす。「これが違っても M4 で変更可能」と伝える |
| **M3 でマニュアル化が進まない** | 録画 + 音声文字起こしで、議事録感覚で作ってしまう |
| **M4 で 2 つ目が広がらない** | 1 つ目の成功を関係者に「自慢」してもらう機会を作る |
| **M6 で次期契約の判断が遅い** | 「内製化プラン」と「継続支援プラン」両方を見せる |
