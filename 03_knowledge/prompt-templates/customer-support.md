# 問い合わせ対応プロンプト集

## 一次回答ドラフト生成

```
あなたは [CLIENT_NAME] のカスタマーサポート担当です。
以下の問い合わせに対する一次回答ドラフトを作成してください。

【問い合わせ】
（本文を貼り付け）

【参考情報】
- 弊社サービス: ...
- 該当しそうなFAQ: ...

【出力】
1. 想定される問い合わせ意図（3案）
2. 各意図に対する回答ドラフト
3. 追加で確認すべき情報
4. 担当部署への引き継ぎが必要かの判断

※ 必ず人間がレビューしてから返信。AI回答をそのまま送らない。
```

## FAQ 整備

```
以下の過去の問い合わせと回答ペアから FAQ を構成してください。

【入力】
（問い合わせ・回答のペアを5〜20件）

【処理】
1. 類似の問い合わせをグルーピング
2. 各グループの代表質問を1つに整理
3. 回答を「結論 → 補足 → 関連リンク」の形式で整形
4. カテゴリ分類（料金・操作・トラブル等）
5. 検索されやすそうな別表現を3つずつ追加
```

## 苦情対応の文章チェック

```
以下の返信文を、苦情対応の観点からレビューしてください。

【返信案】
（本文）

【チェック項目】
1. 共感の表現は適切か
2. 言い訳に聞こえないか
3. 責任範囲を明確にしているか
4. 次のアクションが具体的か
5. 法的リスクのある断定表現はないか

改善案を3つ提示してください。
```

## 多言語対応

```
以下の日本語の問い合わせ回答を、英語に翻訳してください。

【日本語】
（本文）

【条件】
- ビジネスメールとして自然な英語
- 専門用語は業界標準の表現
- 翻訳後、日本語に逆翻訳して意味のズレがないか確認
```
