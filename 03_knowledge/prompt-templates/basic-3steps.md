# 基本プロンプトの3コツ

> AI に頼むときに「最初に教える3つの型」。
> 初心者向けトレーニングでもこの3つだけは必ず伝える。

## コツ1: ざっくり → 具体的に書く

❌ NG例
```
資料作って
```

✅ OK例
```
新規顧客向けの会社紹介資料を作って。
- 対象: 中小企業の経営者
- 目的: 初回商談で配布
- 枚数: 5枚
- トーン: 信頼感重視、堅すぎない
```

## コツ2: AI に役割を与える

❌ NG例
```
このメール返信の文章を書いて
```

✅ OK例
```
あなたは営業アシスタントです。
顧客からの値引き交渉メールに対して、
丁寧かつ譲歩しすぎない返信文を作成してください。
```

## コツ3: 順序立てて指示する

❌ NG例
```
議事録を整理してまとめて要点を出して次のアクションも考えて
```

✅ OK例
```
以下の議事録を3ステップで処理してください:
1. 発言者ごとに発言を整理する
2. 決定事項・宿題事項を箇条書きで抽出する
3. 次回ミーティングまでのアクションを担当者付きで提案する
```

## 応用: 4つ目の補強テクニック

- **形式を指定する**: 「箇条書きで」「表形式で」「Markdown で」
- **長さを指定する**: 「200字以内で」「3行で」
- **避けるべきことを指定する**: 「専門用語は使わないで」「断定表現は避けて」
- **例を見せる**: 「以下のような書き方で:（例文）」
