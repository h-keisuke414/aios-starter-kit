# データ分析プロンプト集

## CSV データの要約

```
以下の CSV データを分析してください。

【データ】
（CSV を貼り付け）

【処理ステップ】
1. データの概要（行数・列数・期間）
2. 各列の意味の推測
3. 異常値・欠損値の指摘
4. 主要な統計量（合計・平均・最大・最小）
5. 気づいた傾向 Top 3

最後に「もっと深く見たい点」を3つ提案してください。
```

## 売上データの傾向把握

```
以下の月次売上データから傾向を読み取ってください。

【データ】
（表を貼り付け）

【出力】
1. 増減傾向（前年比・前月比）
2. 季節性の有無
3. 異常な月とその仮説
4. 来月の予測（範囲で）
5. 注視すべき指標
```

## アンケート自由記述の分類

```
以下の自由記述コメントをカテゴリ分けしてください。

【入力】
（コメント1件1行で羅列）

【処理】
1. 内容を読んでカテゴリを5〜7個提案
2. 各コメントをカテゴリに振り分け
3. カテゴリごとの件数を集計
4. 代表的な意見を1〜2個ずつ抜粋
5. 改善優先度の高いカテゴリを示唆
```

## 表データのクリーニング指示

```
以下のデータをクリーニングするための手順を提案してください。
（実行はしなくてOK、Excelの操作手順 or 数式で示してください）

【現状の問題】
- 表記ゆれ（例: "(株)"と"株式会社"）
- 日付フォーマット混在
- 重複行

【データサンプル】
（先頭10行を貼り付け）
```

## 注意

- AI に元データを直接入れる前に、機密情報の匿名化を必ず実施
- 数値計算は AI でなく Excel / 数式で再検証
- 「分析結果」を鵜呑みにせず、必ず人間が裏取り
