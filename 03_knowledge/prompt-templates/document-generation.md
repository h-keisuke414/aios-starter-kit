# 文書生成プロンプト集

## メール返信

```
以下のメールに対する返信を作成してください。

【受信メール】
（ここに本文）

【返信の方針】
- トーン: 丁寧 / カジュアル / 強気
- 結論: YES / NO / 保留
- 含めたい情報: ...
- 避けたい表現: ...

【出力形式】
件名・宛名・本文の3パートで。
```

## 議事録 → 要約

```
以下のミーティング書き起こしから議事録を作成してください。

【処理ステップ】
1. 発言者ごとの発言を整理
2. 決定事項を箇条書き
3. 宿題事項を「担当者・期限・内容」の表で
4. 次回までのアクションを提案

【書き起こし】
（ここに本文）
```

## 提案書ドラフト

```
あなたはコンサルタントです。
以下の情報を元に提案書のドラフトを作成してください。

【クライアント情報】
- 業種: [INDUSTRY]
- 課題: ...
- 期待する成果: ...

【提案書の構成】
1. 現状認識
2. 課題仮説
3. 解決策3案（A/B/C）
4. 推奨案とその理由
5. スケジュールと費用感

各セクション200〜400字。
```

## 社内向け説明文

```
以下の専門的な内容を、AIをほぼ使ったことがない社員向けに
やさしい言葉で説明してください。

【元の内容】
（ここに本文）

【条件】
- 専門用語には必ず短い注釈
- 例え話を1つ入れる
- 400字以内
```

## 提案メール（営業）

```
あなたは新規開拓の営業担当です。
以下の見込み顧客に対する初回アプローチメールを書いてください。

【顧客情報】
- 業種・規模:
- 想定課題:
- 接点（紹介者・展示会など）:

【メールの方針】
- 売り込みすぎない
- 1回のアポにつなげる
- 200字以内
- 件名は具体的に
```
