# 会議要約プロンプト集

## 録音書き起こし → 議事録

```
以下の会議書き起こしを処理してください。

【書き起こし】
（テキストを貼り付け。長文の場合はチャンク分割）

【処理ステップ】
1. 発言者ごとに発言を整理
2. 議題ごとにセクション分け
3. 決定事項を箇条書き（誰が・何を・いつまでに）
4. 宿題事項を表形式（担当者・期限・内容）
5. 次回までのアクションを提案
6. 議題に関連する追加検討事項を3つ提案

【出力形式】
Markdown / 1500字以内
```

## 短時間ミーティングの要約（15〜30分）

```
以下のミーティングメモを構造化してください。

【メモ】
（殴り書きでOK）

【出力】
- 目的（1行）
- 主な論点（3〜5個）
- 決定事項
- 次のアクション
- 補足: 議論されなかったが重要そうな論点
```

## 営業同席メモ → 商談記録

```
以下の商談メモから、CRMに登録する商談記録を作成してください。

【メモ】
（書き起こし or 手書きメモを貼り付け）

【出力フィールド】
- 商談日時:
- 参加者（先方・こちら）:
- 顧客の課題:
- 提案した解決策:
- 反応・感触:
- 競合の有無:
- 予算感:
- 決裁プロセス:
- 次回アクション:
- 確度（A/B/C）:
- 次回アポ:
```

## マネジメント1on1メモの整理

```
以下の1on1メモを構造化してください。

【メモ】
（本文）

【出力】
1. 部下の気持ち・状態の要約
2. 相談された課題（3つまで）
3. 上司が今後フォローすべきこと
4. 部下に伝えるべきフィードバック
5. 次回1on1で確認したいこと

※ 機密性が高いため AI 投入前に名前等は匿名化すること。
```
