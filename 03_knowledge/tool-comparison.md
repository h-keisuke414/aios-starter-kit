## 選定のフレーム

判定は **上から順に** YES/NO で進める。

### 1. 既存ツール環境

- **Microsoft 365 を使っているか？** YES → Copilot を最優先候補に
- **Google Workspace を使っているか？** YES → Gemini を最優先候補に
- **どちらでもない（Excel ファイルベース・LINE・メール中心）** → ChatGPT or Claude

### 2. データ取り扱い

- **機密・個人情報を入れるか？** YES → Team / Enterprise プラン必須（学習 OFF 設定）
- **顧客名・売上データを匿名化なしで入れる予定があるか？** YES → オンプレ or Azure OpenAI も検討

### 3. 使う人のリテラシー

- **PC が苦手な人が多い** → ChatGPT（UI が一番馴染みやすい）
- **業務効率を本気で詰めたい** → Claude（精度・指示への忠実さ）
- **既存業務フローの中で使いたい** → Copilot / Gemini

### 4. 予算

- **1 人あたり ¥3,000 / 月以下に抑えたい** → ChatGPT Plus or Claude Pro 個別契約
- **5 名以上で使う** → Team プラン（管理画面・SSO 付き）

### 5. 画像・動画生成の必要性

- **SNS 投稿で画像を作りたい** → ChatGPT（DALL·E）or Gemini（Imagen）
- **動画も作りたい** → ChatGPT（Sora）or 別途専用ツール

## 推奨セット例

### パターン A: 業務効率化中心（最も多いケース）

- **メイン**: ChatGPT Team（全社員）¥4,500 / 月 × N 名
- **補助**: Claude Pro（推進担当 1〜2 名のみ）¥3,000 / 月

文書作成・問い合わせ対応・社内 FAQ 中心。

### パターン B: ナレッジ活用中心

- **メイン**: NotebookLM Plus + Gemini for Workspace
- **補助**: Claude Pro（推進担当）

既に Google Workspace。社内マニュアルが膨大で、検索性を上げたいケース。

### パターン C: 既存 M365 活用

- **メイン**: Microsoft 365 Copilot（全社員）¥4,497 / 月
- **補助**: ChatGPT Team or Claude Pro

Outlook・Teams・Excel 中心の業務。営業職が多い会社。

### パターン D: 小規模・コスト最小

- **メイン**: ChatGPT Plus 個別契約（3〜5 名）¥3,000 / 月
- **補助**: NotebookLM 無料版

5 名以下の小規模事業者。月額総額を ¥15,000 程度に抑えたい場合。

### パターン E: AI ネイティブ志向

- **メイン**: Claude Team（全社員）¥3,750 / 月
- **補助**: Perplexity Pro（営業職）¥3,000 / 月

「AI 活用企業」のブランディングを取りに行く会社。後継者主導案件で多い。

## 契約時チェックリスト

- [ ] **データ学習設定 OFF** or 学習されない契約（最重要）
- [ ] ログ保管期間（短い方がリスク低・長い方が監査向き）
- [ ] SSO / SAML 対応の必要性（10 名以上ならほぼ必須）
- [ ] 管理者ダッシュボードの有無
- [ ] 利用ログの取得可否（運用後の効果測定で必要）
- [ ] 月額・年額の差（年額は 10〜20% 安いことが多い）
- [ ] 解約時のデータエクスポート方法
- [ ] 請求は法人カード or 銀行振込か（中小は振込希望が多い）
- [ ] **日本語サポートの有無**（クライアントが不安に思う最大ポイント）

## クライアントへの説明の言い回し

「Claude？ChatGPT？何が違うの？」と聞かれたら:

> 「車に例えると、ChatGPT は **トヨタ**、Claude は **マツダ** のような関係です。トヨタは万人向けで何でもこなせる。マツダは運転好きが選ぶ尖った車。御社の場合、まず万能型の ChatGPT から入り、慣れた人だけ Claude も使う、という入り方が無難です。」

数字や仕様より **比喩で伝える** 方が中小企業の経営者には届きやすい。

## 切り替えのタイミング

- **3 ヶ月使ってみて物足りない** → 補助ツールを足す（メインは変えない）
- **半年使って課題が明確** → メインを再検討
- **新サービスが出た** → 即追従せず 3 ヶ月待って評判を見る

クライアントには「ツールは半年〜1 年で見直す前提」を最初に伝える。
