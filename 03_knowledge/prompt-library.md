# 案件固有プロンプト・ライブラリ

> 運用中に「これは便利」と判断されたプロンプトを蓄積するファイル。
> 案件特有のドメイン用語・フォーマットを含む。

## 使い方

- 新しいプロンプトを追加する際は以下の構造で書く
- 動かなくなった or 改善されたものは削除せず「v2」として追加し、旧版は折りたたむ

## テンプレート

```
### [P-001] プロンプト名

**用途**: 何のために使うか
**使用ツール**: Claude / ChatGPT / Gemini
**作成日**: YYYY-MM-DD
**作成者**:
**バージョン**: v1

#### プロンプト本文

\`\`\`
（プロンプトをここに貼る）
\`\`\`

#### 入力例

\`\`\`
（実際の入力例）
\`\`\`

#### 期待される出力

（出力サンプル）

#### 注意点

- 機密データを入れる場合の匿名化方法
- 失敗ケース
- バージョン履歴
```
