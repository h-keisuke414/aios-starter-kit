# リスクとガバナンス

## 1. データ取り扱いポリシー

### AI に入力してよいデータ

- [ ] 公開情報（自社Webサイト・公開資料）
- [ ] 社内一般情報（マニュアル・規程）
- [ ] 部分的な業務データ（匿名化後）

### AI に入力してはいけないデータ

- [ ] 個人情報（氏名・連絡先・マイナンバー等）
- [ ] 顧客の機密情報
- [ ] 取引先との非公開契約内容
- [ ] 未公開の財務情報
- [ ] 採用候補者情報

### 匿名化ルール

- 氏名 → 「Aさん」「B社」
- 金額 → 桁を保ち下3桁を 0 に
- 日付 → 月単位まで
- 住所 → 市区町村まで
