# 長期機会レーダー

> 「いま提案しないけど、いずれこの会社の役に立ちそうな打ち手」を貯めておく場所。
> ヒアリングを増やすのではなく、**運用しながら気づいたこと** を都度ここに書き溜める。
>
> 提案するタイミングは「相手が困っている」「相手が新しいことを始めたがっている」「信頼が一定以上築けた」など、**機が熟した瞬間** で OK。焦って早く出さない。

## 運用ルール

1. **書く**: 月次レビュー・現場訪問・雑談で気づいたことを 1〜2行でメモ
2. **置く**: 提案候補ごとに「今は出さない」状態でストック
3. **熟成**: 3ヶ月以上寝かせて、まだ意味がありそうなら成熟度を上げる
4. **出す**: 状況が変わった or 相手から相談が来たら、タイミングを見て提案
5. **諦める**: 半年以上動かないものは「Pass」に分類

## 観察の視点

- **困っていそうな業務**: ヒアリング外で見えた手間・属人化・ミス
- **新しいことを始めたがっている**: 経営者の発言、社内の動き
- **業界の追い風**: 法改正・補助金・業界トレンド
- **競合の動向**: 同業他社の取り組み（[../00_discovery/local-research.md](../00_discovery/local-research.md) と連動）
- **AI の進化**: 半年前にはできなかったことが今できる
- **AIOS 他サービスとのつながり**: SNS / LINE / MEO に派生できそうな領域

|---|---|---|---|---|---|
| O-001 | | | 大/中/小 | Seed / Watch / Ready / Pass | |

### 成熟度の定義

- **Seed**: 初観察。具体性なし
- **Watch**: 数回見えてきた。仮説を補強中
- **Ready**: 提案可能なレベル。タイミング待ち
- **Pass**: 当面なしと判断

## エントリーテンプレート

```
### O-XXX: [機会タイトル]

- **観察日**: YYYY-MM-DD
- **きっかけ**: 誰の何の発言・出来事か
- **想定価値**: 何が嬉しくなるか（1〜2文）
- **想定アプローチ**: AI / SNS / LINE / MEO / HP / 別サービス
- **必要な前提**: 信頼関係・予算・社内体制
- **成熟度**: Seed
- **次のアクション**: もう少し観察 / 関連質問を1つだけ次回入れる
- **タイミング目安**: 半年後 / 来期予算検討時 / 相手が相談してきた時
```

## 観察を漏らさないための小さなチェックリスト

毎月の月次レビュー（[../05_operations/monthly-report-template.md](../05_operations/monthly-report-template.md)）作成時にこの問いに答える:

- [ ] 今月、想定外に時間がかかっていそうな業務を見たか？
- [ ] 経営者・推進者が「やりたい」と言ったが今のスコープに入っていないものはあるか？
- [ ] 業界・地域で新しい動き（補助金・規制・競合）はあったか？
- [ ] AIOS 他サービス（SNS / LINE / MEO）との接点で見えたものはあるか？

これだけで OK。深掘りは不要。**気づきの種を1〜2行ストックする** 行為を続けること自体が成果。

## 提案するタイミングの判断基準

提案して良いシグナル:
- 相手が「最近こんなことに困っていて…」と相談してきた
- 業界に明確な変化があり、対応が遅れるリスクが顕在化
- 既存スコープが安定し、次の話題が必要になった（M5〜M6 あたり）
- 信頼関係が築かれ、雑談ベースで打ち手の話ができる関係になった

提案を控えるべきシグナル:
- 既存スコープがまだ揺れている（M1〜M3）
- 相手が忙しすぎる時期（決算・繁忙期・人事異動直後）
- 社内政治が複雑（直近で揉めている案件あり）
- まだ ROI が見えていない（M2 終了前）

## 既出の典型機会（参考リスト）

中小企業でよく見つかる「いずれ役に立つ打ち手」のサンプル。
案件で観察したものを O-XXX として登録していく:

- お客様データの整理・統合（CRM 化）
- LINE 公式アカウントでの再来店促進
- Google ビジネスプロフィール（MEO）強化
- SNS での採用ブランディング
- 採用面接の同席ログ → AI 要約
- 売上データの月次自動可視化
- 業務マニュアルの動画化（社内引き継ぎ）
- 補助金申請文書の AI 下書き
- 多店舗・多拠点への横展開
- 顧客アンケートの自動分析
